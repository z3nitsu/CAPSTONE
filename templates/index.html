About Project

This project is the capstone project for the spring semester of 2022. It uses machine learning algorithms (random forest, k-nearest neighbours, logistic regression, Naive Bayes) and a dataset of patient data from Kaggle to model and predict the existence of various diseases in patients, such as heart disease and diabetes.
